TeaFramework README
---------------------------------------------------------------------

Thank you for downloading TeaFramework.

------------
REQUIREMENTS
------------
1. You will need a JDK (Java Development Kit) 1.6 or above.  

2. Maven 3.0  for build TeaFramework , You can download it from http://maven.apache.org

3. Eclipse for development http://www.eclipse.org


-------------------
BRIEF  GUIDE
-------------------

In command line environment

Build the Project
mvn compile

Test the Project
mvn test

Package the Project
mvn package

Generating the Site
mvn site

Git setup Sample

git config --global user.name "Peter Cheng"
git config --global user.email teaframework@gmail.com
        
Next steps:

  git add README
  git commit -m 'first commit'
  git remote add origin git@github.com:agilesource/teaframework.git
  git push -u origin master
